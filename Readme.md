# TUI Music player
## Description
A TUI music player made for fun and for learning of the ncurses lib.
## Prerequisites
You have following installed system libs on your linux machie:
- libncurses
- libao
- libmpg123
## Build
```bash
make
```
## Run
- you can find executable after build under `./bin/app` path
- load your playlist into `./assets/` directory, all tracks should be into mp3 format 
## Keybinds
- k/j -- move up/down the list
- p -- play selected track
- q -- quit
